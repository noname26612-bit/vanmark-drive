# Запуск разработки vanmark-drive

Эта папка — готовый «посевной» репозиторий: CLAUDE.md (правила агента), docs/ (PRD, архитектура, roadmap; отложенные наработки по 1С — в docs/archive/), .claude/skills/ (4 скилла, агент применяет их сам). Код появится в этапе 0.

## Перед первым запуском (один раз)

Должны быть установлены: **Node.js LTS** и **pnpm** (`npm i -g pnpm`), **Docker Desktop** (для PostgreSQL локально), **git**, **Claude Code**. Если чего-то нет — первый же промпт ниже попросит Claude Code проверить и помочь поставить.

## Как запускать

Терминал → `cd` в эту папку → команда `claude`.

### Промпт №1 (этап 0 — каркас)

```
Проверь, что окружение готово (node, pnpm, docker, git) — если чего-то не хватает, помоги установить.
Затем прочитай CLAUDE.md, docs/PRD.md, docs/ARCHITECTURE.md, docs/ROADMAP.md.
Начинаем Этап 0 из ROADMAP: git-репозиторий и каркас проекта строго по ARCHITECTURE.
Перед установкой зависимостей проверь в вебе актуальные стабильные версии стека (Next.js, Prisma, Auth.js, Serwist, Tailwind).
По завершении прогони критерии приёмки этапа 0 и покажи мне, как запустить приложение и убедиться, что всё работает.
```

### Дальше — по одному этапу за раз

```
Прочитай CLAUDE.md и docs/ROADMAP.md. Делаем Этап N по скиллу feature-dev.
По завершении: критерии приёмки, security-check (если менялись API/БД), и покажи мне, что и где смотреть.
```

К этапу 2 понадобится последний номер задачи из Telegram-чата (для старта нумерации) — просто скажи его агенту. К этапу 7 — VPS и домен (например, drive.van-mark.ru).

## Правила работы (для тебя)

- Один этап = одна сессия. Не проси «сделай всё сразу» — приёмка по этапам и есть контроль качества.
- Принимай этап как пользователь: зайди как Милена и как водитель (логины даст сид этапа 1), прокликай сценарий руками.
- Продуктовые решения — твои: агент обязан спрашивать про логику, а технику решать сам. Если он спрашивает техническое — отвечай «реши сам в рамках ARCHITECTURE».
- Захотелось новую фичу — не вставляй в текущий этап, говори «добавь в бэклог фазы 3 в ROADMAP».
